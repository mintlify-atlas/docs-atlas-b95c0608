## What is Proone?

Proone is a Linux worm designed to target unconfigured IoT embedded devices with MMU. It features self-contained breaking and entering, replicating, IPv6 support, and CNC using DNS over TLS.

This project is a reengineered version of Mirai, developed with a serious approach and additional capabilities. The original goal was "pruning" poorly secured devices from the Internet, though the project has been released as open-source research instead.

## Core Subsystems

Proone consists of four main subsystems that work organically to infect hosts and provide CNC capabilities:

<Steps>
  <Step title="Heartbeat">
    Backdoor and CNC mechanism using DNS TXT records or TCP/IP connections for command and control
  </Step>
  <Step title="Recon">
    Network reconnaissance worker that discovers vulnerable nodes on the Internet and local networks using raw socket packet crafting
  </Step>
  <Step title="BNE (Break and Enter)">
    Exploitation worker that uses credential brute force and vulnerability exploits to compromise target hosts
  </Step>
  <Step title="Resolv">
    Custom DNS over TLS resolver designed specifically for secure CNC communications
  </Step>
</Steps>
